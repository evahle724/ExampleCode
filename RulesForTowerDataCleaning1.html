Rule 1: Windspeed¶

Writing the Rule as a Function¶

If the average windspeed of any given ten minute time interval is less than or equal to 0$\frac{\text{m}}{\text{s}}$, it will be flagged and considered bad data. If it is more than 90$\frac{\text{m}}{\text{s}}$, it will also be flagged and considered bad data.

This code defines a function for Rule 1 that files through the columns of data, selects all of the speed average columns, and creates a new column that returns 'True' if the average speed value is outside the range of 0 to 90 and returns 'False' if the data is inside the range.

In [2]:
def rule_1_flag(df):
    for col in df.columns:
        if re.match('SPD_.*_AVG', col) is not None:
            df['FLAG_SPD_' + str(col.split('_')[1]) + '_' + str(col.split('_')[2]) +
               '_AVG'] = (df[[col]]<0) & (df[[col]]>=90)

Now we will call the function and apply it on our Tower B data. Once it has been called, we can look at the data and see that new flag columns have been added for each average speed column.

Rule 2: Flatlining¶

Writing the Rule as a Function¶

For a certain time, if the average wind speed minimum absolute difference is less than than 0.01$\frac{\text{m}}{\text{s}}$, it will be flagged as flatlining and considered bad data.

This code rolls through the 5 previous entries in a column and takes the maximum absolute difference between any of those 5 entries. It then checks to see if this difference is less than 0.1$\frac{\text{m}}{\text{s}}$. The statement inside df.apply() states that "raw = True" which has to do with a warning message about updates to df.apply() and this using this statement will keep the function of df.apply() the same even after updates have happened.

In [6]:
def rule_2_flag(df):
    for col in [col for col in df.columns if re.match('SPD_.*_AVG', col)]:
        df['FLATLINE_' + str(col.split('_')[1]) + '_' + str(col.split('_')[2])] \
        = df[col].rolling('50min').apply(lambda x: min(abs(x-x[-1])), raw = True)<0.1

Now we will call the function and apply it on our Tower B data. Once it has been called, we can look at the data and see that new flag columns have been added for each average speed column.

Rule 3: Wind Vane Direction¶

For a certain time, if the average wind direction maximum absolute difference is greater than 45$^\circ$ and any average wind speed is greater than 3$\frac{\text{m}}{\text{s}}$, it will be flagged for wond vane direction agreement and considered bad data.

This code rolls through the 10 previous entries in a column and takes the maximum absolute difference between any of those 10 entries. It then checks to see if this difference is greater than 45$^\circ$ and if any speed is greater than 3$\frac{\text{m}}{\text{s}}$. The statement inside df.apply() states that "raw = True" which has to do with a warning message about updates to df.apply() and this using this statement will keep the function of df.apply() the same even after updates have happened.

In [10]:
def rule_3_flag(df):
    SPDCOLS = [col for col in df.columns if re.match('SPD.*_AVG', col) is not None]
    SPDdf = df[SPDCOLS]>3
    for col in [col for col in df.columns if re.match('DIR_.*_AVG', col)]:
        df['FLAG_VANE_DIR_' + str(col.split('_')[1]) + '_' + str(col.split('_')[2])] \
        = df[col].rolling('100min').apply(lambda x: max(abs(x-x[-1])), raw = True)>45 \
        * SPDdf.any(axis = 'columns')

Now we will call the function and apply it on our Tower B data. Once it has been called, we can look at the data and see that new flag columns have been added for each average vane direction column.

Rule 4: Tower Shading¶

For a point when the ratio of average windspeeds at the same height is at a maximum or a minimum, the speed will be flagged when the average vane direction is $\pm$30$^\circ$ from the average vane direction at that point.

This code runs through the average speed columns and then takes the ratio between columns that have the same height. It then computes the index of the maximum ratio between the two columns. This index is then used to find the vane direction angle at that same time. The speed column is then flagged at the times when the average vane diection is within $\pm$30$^\circ$ of the angle found.

In [14]:
def rule_4_flag(df):
    SPDAVG = [col for col in df.columns if re.match('SPD_.*_AVG', col) is not None]
    DIRCOLS = [col for col in df.columns if re.match('DIR_.*_AVG', col)]
    for col1 in SPDAVG:
        df['SHADEFLAG_' + str(col1.split('_')[1]) + '_' + str(col1.split('_')[2])] = False
        for col2 in SPDAVG:
            if col1 != col2 and col1.split('_')[1] == col2.split('_')[1]:
                row = (df[col2] / df[col1]).idxmax()
                df['MEAN_DIR'] = df[DIRCOLS].mean(axis = 1)
                angle = df['MEAN_DIR'][row]
                df['SHADEFLAG_' + str(col1.split('_')[1]) + '_' + str(col1.split('_')[2])] = abs(df['MEAN_DIR']-angle)<30

Now we will call the function and apply it on our Tower B data. Once it has been called, we can look at the data and see that new flag columns have been added for each average speed column.
